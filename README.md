Sketch Make Long Shadow
==========================

Create long shadows quickly and easily.

1. Select a Shape layer and run the plugin.
2. Set the distance and direction of the shadow.
3. Hit `Make Shadow`

_Note: The plugin only supports Shape Layers. So to create long shadows for Text Layers, be sure to Convert Text to Outlines (`âŒ˜` + `Shift` + `O`)._
